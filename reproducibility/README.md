# WG-SRC Reproducibility Package

This repository contains the final reproducibility materials for WG-SRC: a white-box graph signal-subspace ridge classifier and diagnostic probe.

## Final paper datasets

The final paper uses exactly six datasets:

- amazon-computers
- amazon-photo
- chameleon
- cornell
- texas
- wisconsin
