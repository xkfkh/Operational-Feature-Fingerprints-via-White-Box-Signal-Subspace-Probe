whitebox_gcn2.py (Agent 4) experiment results
============================================================

Method: ReduNet full unrolled gradient ascent + graph aggregation + ISTA

Architecture:
  - Input projection (linear, xavier init)
  - 2 x ReduNetGCNLayer (LayerNorm + E^l + C_k^l + A_norm agg + ISTA)
  - Output classifier (linear)

Layer math:
  E^l   = (I + alpha/n * H^T H)^{-1}
  C_k^l = (I + alpha/n_k * H_k^T H_k)^{-1}
  H'    = H + eta*(H @ E - A_norm @ sum_k(gamma_k*(H*pi_k) @ C_k))
  H_out = ISTA(H', threshold)

Loss = CE - lambda_mcr * DeltaR + lambda_orth * L_orth

Key insight: DeltaR >> CE, so lambda_mcr must be tiny (< 1e-4)
